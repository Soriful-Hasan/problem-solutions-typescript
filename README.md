<h1>টাইপ স্ক্রিপ্ট ব্লগ</h1>

২। keyof কি ?
keyof হলো TypeScript এর একটি কীওয়ার্ড এই কীওয়ার্ড খুব এই দরকারি। যখন আমরা অবজেক্ট এর সমস্ত কী গুলা থেকে টাইপ তৈরী করতে চাই। 
তখন আসলে আমাদের keyof কীওয়ার্ড টি দরকার পরে । এবং আমরা খুব সহজে এই কীওয়ার্ড এর মাধ্যমে অবজেক্ট এর কী দিয়ে ইউনিয়ন টাইপ তৈরী করতে পারি 
এবং আমরা টাইপ হিসাবে প্রয়োজন অনুযায়ী ব্যবহার করতে পারি। 

উদাহরণ => 
type User = {
  name: string
  age: number
  isActive: boolean
}
type UserKeys = keyof User
// "name" | "age" | "isActive"

এই উদাহরণ দিয়ে আমরা বুজতে পেরেছি typeof দিয়ে আসলে একটা অবজেক্ট থেকে কী গুলা কে একটা ইউনিয়ন টাইপ তৈরী করে। 


৩। any , unknown , never টাইপ এর মধ্যে পার্থক কি ? 
any টাইপ মানে হলো TypeScript এ টাইপ এর সাধীনতা দেয়া। যেমন আমরা যদি any টাইপ দিয়ে কোনো ভ্যারিয়েবল ডিফেন্ড করি তখন সেই ভ্যারিয়েবল এ যে কোনো মান রাখা যাবে। 
আমরা চাইলে স্ট্রিং , নম্বর , বুলিয়ান ইত্যাদি - যে কোনো টাইপ রাখতে পারবো। 
উদাহরণ :- আমি এমন একটা ভ্যারিয়েবল নিয়েছি সেই ভ্যারিয়েবল এ যদি আমি any টাইপ দেয় তাহলে আমি সেই ভ্যারিয়েবল এ যে কোনো টাইপ এর ম্যান রাখতে পারবো। 

unknown টাইপ হলো সহজ বাংলায় অজানা টাইপ। মানে আমি এমন টাইপ দিব যেটা আসলে আমার ফাঙ্কশন বা ভ্যারিয়েবল আগে থেকে জানবে না যে কি টাইপ এর ডাটা আসবে।
উদাহরণ :- আমি যদি একটা ফাঙ্কশন লিখি তখন আমি জানি না যে আমার ফাংশন টি কোন টাইপ রিটার্ন করবে তখন আমি unknown টাইপ ডিফাইন্ড করে দিতে পারি। 

never হলো এমন একটা টাইপ যেই টাইপ আসলে কখনোই এমন কিছু নয় যা ঘটতে পারে না। আমরা যদি কোনো ফাঙ্কশন এ যদি কোনো কিছু রিটার্ন না করে বা কখনো কিছু  রিটার্ন না করে 
তখন never টাইপ আসলে ডিক্লেয়ার করে দেয়া যাই। 
উদাহরণ :-
function crash(): never {
  throw new Error("Something went wrong");
}
